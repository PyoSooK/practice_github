# 기본 문법

## Headings

HTML : <h1> ~ <h6>

Markdown : 

1. **#의 개수 (6단계)**
    
    # 뒤는 항상 뛰어 써야 하며 heading 앞 뒤로 뛰어 써야 한다.
    
    <aside>
    
    # Heading 1
    
    ## Heading 2
    
    ### Heading 3
    
    #### Heading 4
    
    ##### Heading 5
    
    ###### Heading 6
    
    </aside>
    
    # Heading 1
    
    ## Heading 2
    
    ### Heading 3
    
    #### Heading 4
    
    ##### Heading 5
    
    ###### Heading 6
    

1. == 혹은 - - 사용 (상위 2단계 )
    
    <aside>
    
    Heading 1
    
    ========
    
    Heading 2
    
    -----------
    
    </aside>
    
    Heading 1
    ========
    
    Heading 2
    -----------
    

##
